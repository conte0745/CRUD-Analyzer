Spring Boot + MyBatis プロジェクトの Controller → Service → Mapper の呼び出し関係を解析し、 エンドポイントとテーブルのCRUD操作を可視化するツールです。

基本的な使用方法

// 1. 設定ファイルの作成
cp analyzer-config.properties.example analyzer-config.properties

// 2. 設定ファイルの編集（パスを実際のプロジェクトに合わせて変更）
src.directory=/path/to/your/project/src/main/java
resources.directory=/path/to/your/project/src/main/resources
output.directory=/path/to/your/project/docs

// 3. 実行（引数不要）
java -jar crud-analyzer-all.jar

プログラムからの使用

// 設定ファイルから自動読み込み
var analyzer = new CrudAnalyzer();
var result = analyzer.analyze();

// 結果の取得
List<Endpoint> endpoints = result.endpoints();
List<SqlMapping> sqlMappings = result.sqls();

設定ファイル

analyzer-config.properties で動作をカスタマイズできます： 

src.directory=/path/to/src/main/java
resources.directory=/path/to/src/main/resources
output.directory=/path/to/output
analysis.include.generated=true
analysis.include.dynamic.sql=true

詳細な使用方法とサンプルについては、プロジェクトのREADME.mdを参照してください。
